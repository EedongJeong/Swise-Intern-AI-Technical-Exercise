The code performs four main tasks:

1. Receive Device Data 

2. Build an LLM Prompt 

3. Simulate an OpenAI API Call 

4. Display the Security Insight 

How to Run:

Execute in any terminal or command line:

python LocalAgentPractical.py

Utilising OpenAi API Key:

1. Retrieve API key from OpenAI website
2. Input setx OPENAI_API_KEY "sk-proj-abc123..." in any terminal
